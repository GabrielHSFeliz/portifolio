## 📖 Sobre

Este repositório contém o código-fonte do meu site de portfólio — uma página única (`index.html`), sem frameworks e sem build step, feita para apresentar quem eu sou, minha stack e os projetos que já publiquei no GitHub.

O design segue uma identidade visual dark purple, com elementos de glassmorphism, órbitas animadas e cards flutuantes, inspirado em referências de portfólios modernos.

## ✨ O que tem na página

- **Hero** com apresentação, chips de tecnologias orbitando um núcleo animado e link direto para o GitHub.
- **Sobre mim** — minha trajetória como desenvolvedor júnior.
- **Stack** — ferramentas que uso no dia a dia, organizadas por categoria (back-end, front-end, dados, qualidade, plataforma).
- **Projetos** — grid com os principais repositórios publicados, com descrição, status e link direto.
- **Tecnologias** — logos das linguagens e frameworks que utilizo.
- **Contato** — e-mail, LinkedIn, GitHub e telefone.

Totalmente responsivo (mobile, tablet e desktop) e sem dependências externas de build — é só abrir o `index.html`.

## 🚀 Como rodar localmente

Não tem build, não tem `npm install`. Só clonar e abrir:

```bash
git clone https://github.com/GabrielHSFeliz/portifolio.git
cd portifolio
```

Depois é só abrir o `index.html` direto no navegador, ou usar uma extensão como *Live Server* no VS Code para recarregar automaticamente ao editar.

## 🌐 Deploy

Este site está publicado com **GitHub Pages**, direto a partir da branch `main`:

1. `Settings` → `Pages`
2. Source: `Deploy from a branch`
3. Branch: `main` / `root`

Qualquer push para o `index.html` na `main` atualiza o site automaticamente em alguns minutos.
